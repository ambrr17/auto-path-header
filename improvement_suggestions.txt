# Предложения по улучшению проекта auto-path-header
# Автор: Niklis

1. Настройки формата и исключений:
   - formatTemplate для кастомного вида комментария (напр. "// [path]").
   - disabledLanguages: список языков, для которых функционал отключён.
2. Поведение для файлов с несколькими строками:
   - режим вставки: автоматически / по запросу / никогда.
3. CI/CD (GitHub Actions):
   - сборка, unit+integration тесты, публикация артефакта VSIX.
4. Качество кода:
   - ESLint + Prettier, покрыть JSDoc публичные API.
5. Документация (остатки):
   - примеры (код/скриншоты), расширить раздел "Сценарии" и "Contributing".
