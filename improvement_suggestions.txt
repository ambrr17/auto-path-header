# Предложения по улучшению проекта auto-path-header
# Автор: Niklis

1. Пользователь может задать кастомный шаблон комментария для конкретного **расширения файла** (включая составные, например `.env.local`).

Приоритет:
- customTemplatesByExtension[расширение]
- formatTemplate (если задан)
- Стандартный шаблон языка ({comment})
- Расширение определяется через path.extname(filePath) (включая точку), регистронезависимо.
- Пользователь может задать кастомный шаблон комментария для конкретного **расширения файла** (включая составные, например `.env.local`).

Формат настройки:
  ```json
  "autoPathHeader.customTemplatesByExtension": {
    ".env.local": "# LOCAL OVERRIDE — {path}",
    ".test.ts": "// 🧪 TEST: {path}",
    "Dockerfile.dev": "# DEV BUILD: {path}"
  }
  ```
